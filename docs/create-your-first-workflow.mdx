# Create Your First Workflow

To add a workflow to your newly created application:

1. Navigate to the **Applications** section from the main menu.
2. Expand the **Application** folder to display its child folders: Workflows, Schemas, and Resources.
3. Expand the **Workflows** folder.
4. Click the **Add a Workflow** link.\
   The Create Workflow modal appears.
5. Click **New**. Your workflow is now created and ready for configuration.
