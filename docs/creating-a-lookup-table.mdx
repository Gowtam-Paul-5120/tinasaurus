# Creating a Lookup Table

To create a Lookup table, follow these steps

Click on the + icon on the Lookup Table side panel. From the context menu, you have two options

- &#x20;**Create from Blank Table**: Start with a fresh, blank lookup table.

</figure>

- &#x20; **Create from CSV:** If you already have a CSV file, you can directly import it.

<figure>
  <img src="/img/assets/Lookup-CSV.gif" alt=""></img>
  <figcaption></figcaption>
</figure>

### **2. Configure Lookup Table**

Fill in Name and Description

- Define Table Entries
- You can configure Lookup Table as either Default Storage or Vault.
- Add rows and Columns in Lookup Table.

  <img src="/img/assets/Lookup-configure.gif" alt=""></img>
  <figcaption></figcaption>
</figure>
